ABOUT PROJECT

Algorithm description: We want to develop a machine learning algorithm that can be used by (American) football coaches and newscasters in real time to predict the outcome of a football play. Using independent variables such as down, distance, score margin, team, and time remaining in the game, we want to predict whether a play will result in a pass, kick, or run. We plan on developing 3-4 ML models that range from simple (Logistic Regression) to more complex (Neural Network), and will report the results in performance metrics for each algorithm. Application of algorithm: The goal of this algorithm is to provide real-time predictions for football coaches and newscasters as to which type of play is most likely to be run. This can be especially useful for coaches because teams try to predict what the opponent is going to do while they are playing defense, so that they can effectively counter it. Coaches traditionally rely on their intuition to make this prediction, but if they have an ML-based model telling them what their opponent is (most likely) about to do, they can make the right decision more often, and properly align their defense to counter the upcoming type of play. Broadcasters could also make use of a model like this because it would help them more accurately tell their audience how a game is going to unfold while they are calling it. Similar to coaches, using an ML model to make these predictions would allow for more accuracy than using their intuition alone.
